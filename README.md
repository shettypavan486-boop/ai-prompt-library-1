# AI Prompt Library Application

A full-stack web application to create, view, and manage AI prompts with complexity levels and view tracking.

---

## 🚀 Tech Stack

### Frontend
- Angular (Standalone Components)
- TypeScript
- HTML/CSS

### Backend
- Django
- Django REST Framework

### Database & Cache
- PostgreSQL
- Redis (for view count tracking)

### DevOps
- Docker
- Docker Compose

---

## 📦 Features

- View all prompts
- Add new prompts
- View detailed prompt information
- Track prompt view count using Redis
- Full Dockerized setup (Frontend + Backend + DB + Redis)
